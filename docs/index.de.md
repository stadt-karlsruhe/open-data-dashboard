## Befehle

* `mkdocs new [dir-name]` - Neues Projekt erstellen.
* `mkdocs serve` - Development Server mit live-reload starten.
* `mkdocs build` - Dokumentation bauen.
* `mkdocs -h` - Hilfemeldung drucken und beenden.
